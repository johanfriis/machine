TMUX
######

## tmux commands

$ tmux attach (ta)

## misc keybinds

  d     detach
  t     big clock
  :     prompt
  ?     help

## window keybinds

  c     create window
  w     list windows
  n     next window
  p     previous window
  f     find window
  ,     name window

## pane keybinds

  %     vertical split
  "     horizontal split
  o     swap panes
  x     kill pane
  ⍽     (space) toggle layout
  z     toggle zoom pane

## copy-mode (VI)

# upon entering copy-mode, vi style, common vim commands work, unprefixed

  [     start copy-mode
  ⍽     (space) start selection
  ↵     (enter) copy selection
  ⎋     (escape) clear selection
  :     goto line
  q     quit copy-mode
